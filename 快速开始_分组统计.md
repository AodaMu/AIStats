# 🚀 快速开始：生成分组描述统计表

## 📋 3分钟生成你的表格

### **第1步：启动应用（10秒）**
双击运行：
```
启动应用.bat
```
等待浏览器自动打开（通常是 http://localhost:8501）

---

### **第2步：导入数据（20秒）**
1. 点击左侧 **📁 数据视图**
2. 点击"选择数据文件"
3. 选择 `666.xlsx`
4. 看到 ✅ 成功导入数据

---

### **第3步：选择分组统计（10秒）**
1. 点击左侧 **📊 统计视图**
2. 在"选择统计方法"下拉菜单中
3. 选择 **📊 分组描述统计**（第2个选项）

---

### **第4步：配置参数（30秒）**

#### 基本设置：
- **选择分组变量**：选择 `class`（年级）
- **选择要分析的变量**：选择 `1.1, 1.2, 1.3`（维度1的题目）

#### 高级设置：
- 点击展开 **🔧 高级选项：维度得分计算**
- ✅ 勾选 **"计算维度得分"**

---

### **第5步：生成结果（5秒）**
点击 **"计算分组统计"** 按钮

你会看到类似这样的表格：

```
┌────────┬────────┬──────┬────────┬────────┬────────┐
│ class  │ 样本量 │ 均值 │ 标准差 │ 最小值 │ 最大值 │
├────────┼────────┼──────┼────────┼────────┼────────┤
│ 6      │ 160    │ 3.4  │ 0.8    │ 1.0    │ 5.0    │
│ 7      │ 246    │ 3.6  │ 0.9    │ 1.3    │ 5.0    │
│ 8      │ 217    │ 3.3  │ 0.8    │ 1.0    │ 5.0    │
│ 9      │ 287    │ 3.6  │ 0.9    │ 1.0    │ 5.0    │
│ 10     │ 158    │ 3.6  │ 0.8    │ 1.3    │ 5.0    │
│ 11     │ 107    │ 3.6  │ 0.8    │ 1.7    │ 5.0    │
└────────┴────────┴──────┴────────┴────────┴────────┘
```

---

### **第6步：下载结果（5秒）**
在结果表格下方：
- 点击 **📥 下载 Excel** 按钮
- 文件会保存为 `分组统计_class.xlsx`

---

## 🎯 生成完整的5维度表格

重复步骤4-6，共5次：

| 次数 | 选择的变量 | 维度名称 | 保存文件名 |
|------|-----------|---------|-----------|
| 1    | 1.1, 1.2, 1.3 | 维度1 | 维度1.xlsx |
| 2    | 2.1, 2.2 | 维度2 | 维度2.xlsx |
| 3    | 3.1, 3.2, 3.3 | 维度3 | 维度3.xlsx |
| 4    | 4.1, 4.2 | 维度4 | 维度4.xlsx |
| 5    | 5.1, 5.2 | 维度5 | 维度5.xlsx |

**每次记得勾选"计算维度得分"！**

---

## 📊 合并成最终表格

在 Excel 中：
1. 打开 `维度1.xlsx`
2. 复制其他4个文件的"均值"和"标准差"列
3. 粘贴到 `维度1.xlsx` 的右侧
4. 调整列名为蒙古语（如需要）
5. 完成！

---

## 💡 小技巧

### **一次性查看所有题目**
如果不勾选"计算维度得分"，直接选择所有题目：
- 选择：`1.1, 1.2, 1.3, 2.1, 2.2, 3.1, 3.2, 3.3, 4.1, 4.2, 5.1, 5.2`
- 结果：会显示每个题目的均值和标准差（列会很多）

### **按学校分组**
- 分组变量改为：`school`
- 可以看到不同学校的统计结果

### **查看成绩分布**
- 分组变量：`class`
- 分析变量：`ch1, ch2, ch3, ch4, ch5`
- 查看各年级的成绩情况
