## 📝 双语格式规范

**格式**：中文 / 西里尔蒙文

**示例**：
```python
st.title("数据视图 / Өгөгдлийн харагдац")
st.button("保存 / Хадгалах")
st.info("提示信息 / Мэдээлэл")
```
