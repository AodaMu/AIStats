## 🎯 标签1：描述统计 - 详细化内容

### 1. 均值
```
原来：5个学生成绩：60, 70, 80, 90, 100 → 均值 = 80分

现在：
5个学生成绩：60, 70, 80, 90, 100
均值 = (60+70+80+90+100) / 5 = 400 / 5 = 80分

应用：
- 代表班级的平均水平
- 用于比较不同班级的成绩
- 受极端值影响较大
```

### 2. 标准差
```
原来：班级A的标准差=5分，班级B的标准差=15分 → B班成绩波动更大

现在：
班级A成绩：78, 79, 80, 81, 82（紧凑）
班级B成绩：60, 70, 80, 90, 100（分散）

班级A的标准差 ≈ 1.4分
班级B的标准差 ≈ 15.8分

解释：
- B班标准差大 → 成绩波动大 → 学生水平差异大
- A班标准差小 → 成绩波动小 → 学生水平比较接近
```

### 3. 中位数
```
原来：数据：10, 20, 30, 40, 50 → 中位数 = 30

现在：
例子1：奇数个数据
数据：10, 20, 30, 40, 50
排序后中位数 = 30（第3个）

例子2：偶数个数据
数据：10, 20, 30, 40
排序后中位数 = (20 + 30) / 2 = 25

应用：
- 不受极端值影响
- 适合有离群值的数据
```

### 4. Q1和Q3
```
原来：100个学生中，Q1=70分 → 25个学生成绩≤70分

现在：
100个学生成绩排序后：
Q1 = 70分

解释：
- 25个学生成绩 ≤ 70分
- 75个学生成绩 > 70分
- 表示成绩较低的学生的分界线

四分位数范围（IQR）= Q3 - Q1 = 85 - 70 = 15分
```

### 5. 最大值和最小值
```
原来：班级成绩：60, 70, 80, 90, 100 → 最大值 = 100分

现在：
班级成绩：60, 70, 80, 90, 100
最大值 = 100分
最小值 = 60分

数据范围 = 最大值 - 最小值 = 100 - 60 = 40分

应用：
- 了解数据的上下限
- 检测异常值
- 评估数据的跨度
```

### 2. t值
```
原来：班级A平均80分，班级B平均75分，t=2.5 → 差异较大

现在：
班级A：平均80分，标准差5分，20个学生
班级B：平均75分，标准差6分，25个学生

计算SE = √(5²/20 + 6²/25) = √2.69 ≈ 1.64
计算t = (80 - 75) / 1.64 ≈ 3.05

|t|值越大 → 差异越明显
t = 3.05表示差异很大
```

### 3. 显著性
```
原来：结果标注为**，表示p<0.01，非常显著

现在：
结果显示：t=2.5, p=0.008**

解释：
- ** 表示p<0.01
- 说明有99%的把握认为两组有显著差异
- 只有1%的概率是由随机误差造成的
```

### 4. 自由度
```
原来：样本量n=30，则df=29

现在：
例子1：单样本t检验
样本量n=30 → df = 30 - 1 = 29

例子2：独立样本t检验
班级A：n₁=20，班级B：n₂=25
df = 20 + 25 - 2 = 43

自由度越大 → t分布越接近正态分布
```

### 5. 置信区间
```
原来：班级平均成绩95% CI: [75, 85]

现在：
班级平均成绩80分，标准误差2分

95% CI = 80 ± 1.96 × 2 = 80 ± 3.92 = [76.08, 83.92]

解释：
- 真实平均成绩有95%的概率在76.08-83.92分之间
- 如果重复抽样100次，约95次的置信区间会包含真实值
```

### 6. 效应量
```
原来：d=0.5表示中等效应

现在：
班级A平均80分，班级B平均75分，合并SD=10

Cohen's d = (80 - 75) / 10 = 0.5

解释：
- d = 0.5表示中等效应
- 即使P值不显著，差异也可能有实际意义
- 效应量反映的是差异的实际大小
```

### 7. Cohen's d
```
原来：d=0.6说明两组差异达到中等效应

现在：
三个研究的效应量对比：

研究1：d = 0.2（小效应）
- 差异存在但很小
- 实际意义有限

研究2：d = 0.5（中等效应）
- 差异明显
- 有一定的实际意义

研究3：d = 0.8（大效应）
- 差异很大
- 实际意义重要
```

## 🎯 标签3：相关与回归 - 详细化内容

### 1. 相关系数(r)
```
原来：学习时间与成绩的r=0.85 → 关系很强

现在：
学习时间与成绩的关系：
- 学生A：学习5小时，成绩80分
- 学生B：学习8小时，成绩88分
- 学生C：学习10小时，成绩95分

计算相关系数 r = 0.95

解释：
- r > 0 表示正相关
- r = 0.95接近1 → 关系很强
- 学习时间越多，成绩越高
```

### 2. 相关矩阵
```
原来：成绩、学习时间、出勤率三个变量的相关矩阵

现在：
三个变量的相关矩阵：

          成绩  学习时间  出勤率
成绩      1.00   0.85    0.72
学习时间  0.85   1.00    0.68
出勤率    0.72   0.68    1.00

解释：
- 成绩与学习时间的相关系数 = 0.85（强正相关）
- 成绩与出勤率的相关系数 = 0.72（较强正相关）
- 学习时间与出勤率的相关系数 = 0.68（中等正相关）
```

### 3. 显著性检验
```
原来：r=0.3, n=100, p=0.003<0.05 → 相关显著

现在：
例子1：强相关且显著
r = 0.85, n = 100
计算 → P = 0.001 < 0.05
结论：相关关系显著

例子2：弱相关且不显著
r = 0.15, n = 30
计算 → P = 0.42 ≥ 0.05
结论：相关关系不显著，可能是随机波动
```

### 4. 回归方程
```
原来：成绩 = 20 + 0.8×学习时间

现在：
一元回归例子：
成绩 = 20 + 0.8×学习时间

解释：
- a = 20（截距）：不学习时的基础成绩
- b = 0.8（斜率）：每多学习1小时，成绩增加0.8分
- 学习5小时 → 预测成绩 = 20 + 0.8×5 = 24分

多元回归例子：
成绩 = 10 + 0.7×学习时间 + 0.5×出勤率
```

### 5. R²
```
原来：预测成绩的R²=0.65 → 拟合度良好

现在：
预测成绩的回归模型：
R² = 0.65

解释：
- 65%的成绩变异可由学习时间解释
- 35%的成绩变异由其他因素造成
- 模型拟合度良好

R²的判断标准：
- R² > 0.7：拟合度很好
- 0.5 < R² < 0.7：拟合度良好
- R² < 0.5：拟合度一般
```

### 6. 调整R²
```
原来：多元回归中，调整R²通常小于R²

现在：
多元回归模型对比：

模型1（1个自变量）：
R² = 0.65，Adjusted R² = 0.64

模型2（5个自变量）：
R² = 0.68，Adjusted R² = 0.62

解释：
- 虽然R²增加了，但调整R²反而下降
- 说明增加的变量没有提高模型质量
- 应该选择模型1
```

### 7. VIF
```
原来：变量A的VIF=2.5 → 无共线性

现在：
多元回归中三个变量的VIF值：

变量A：VIF = 2.5 ✓ 无共线性
变量B：VIF = 6.8 ⚠️ 可能存在共线性
变量C：VIF = 15.2 ✗ 严重共线性

处理方法：
- 变量A可以保留
- 变量B需要检查
- 变量C应该删除或转换
```
