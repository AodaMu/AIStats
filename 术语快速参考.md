# 📚 术语快速参考指南

## 📊 描述统计术语

| 术语 | 含义 | 公式 | 例子 |
|------|------|------|------|
| **均值** | 数据的平均值 | Mean = Σx / n | 成绩：60,70,80,90,100 → 均值=80 |
| **标准差** | 数据离散程度 | SD = √[Σ(x-Mean)²/n] | SD=10表示数据在均值±10范围内 |
| **中位数** | 排序后的中间值 | 第50%位置的值 | 数据：10,20,30,40,50 → 中位数=30 |
| **Q1** | 25%分位数 | 第25%位置的值 | 数据：10,20,30,40,50 → Q1≈17.5 |
| **Q3** | 75%分位数 | 第75%位置的值 | 数据：10,20,30,40,50 → Q3≈42.5 |
| **最大值** | 最大数值 | max(x₁,x₂,...,xₙ) | 数据：10,20,30,40,50 → 最大值=50 |
| **最小值** | 最小数值 | min(x₁,x₂,...,xₙ) | 数据：10,20,30,40,50 → 最小值=10 |

## 🔬 假设检验术语

| 术语 | 含义 | 判断标准 | 例子 |
|------|------|---------|------|
| **P值** | 概率值 | P<0.05显著 | P=0.02<0.05，结果显著 |
| **t值** | t检验统计量 | \|t\|越大差异越大 | t=2.5表示差异较大 |
| **显著性** | 显著性水平 | ***p<0.001, **p<0.01, *p<0.05 | 标注**表示p<0.01 |
| **自由度(df)** | 可自由变化的数据个数 | df=n-1 | n=30，则df=29 |
| **置信区间** | 真实参数范围 | 通常95% | 95% CI: [70,80] |
| **效应量** | 差异的实际大小 | d=0.2(小), 0.5(中), 0.8(大) | d=0.6为中等效应 |
| **Cohen's d** | 效应量指标 | d>0.8大效应 | d=0.6说明差异中等 |

## 🔗 相关与回归术语

| 术语 | 含义 | 范围 | 例子 |
|------|------|------|------|
| **相关系数(r)** | 两变量关系强度 | -1到+1 | r=0.85强正相关 |
| **相关矩阵** | 多变量相关系数表 | - | 3个变量的3×3表格 |
| **显著性检验** | 相关是否显著 | p<0.05显著 | r=0.3, p=0.003显著 |
| **回归方程** | 预测方程 | Y=a+b×X | 成绩=20+0.8×学习时间 |
| **R²** | 模型解释比例 | 0-1 | R²=0.65，65%变异可解释 |
| **调整R²** | 考虑变量个数的R² | 0-1 | 多元回归中<R² |
| **VIF** | 共线性指标 | <5无共线性 | VIF=2.5无共线性问题 |

## 📈 图表类型选择

| 图表类型 | 用途 | 适用场景 | 例子 |
|---------|------|---------|------|
| **柱状图** | 比较数值大小 | 分类数据、频数统计 | 不同班级的平均成绩 |
| **散点图** | 展示两变量关系 | 相关分析、趋势探索 | 学习时间与成绩关系 |
| **折线图** | 显示时间趋势 | 时间序列、趋势分析 | 股票价格随时间变化 |
| **箱线图** | 展示分布情况 | 多组比较、异常值检测 | 不同班级成绩分布 |
| **饼图** | 展示比例关系 | 组成比例、百分比 | 各类别销售占比 |
| **直方图** | 展示分布形状 | 正态性检验、分布分析 | 学生成绩分布 |
| **热力图** | 展示矩阵数据 | 相关矩阵可视化 | 变量间相关系数 |

## 🎯 高级分析方法

| 方法 | 比较对象 | 前提条件 | 适用场景 |
|------|---------|---------|---------|
| **单样本t检验** | 样本 vs 总体 | 数据正态 | 检验平均值 |
| **配对样本t检验** | 同组前后 | 差值正态 | 重复测量 |
| **独立样本t检验** | 两个独立组 | 正态+方差齐 | 两组比较 |
| **单因素ANOVA** | 多个独立组 | 正态+方差齐 | 多组比较 |
| **Levene检验** | 方差齐性 | - | 检验方差相等 |
| **Cronbach's Alpha** | 量表一致性 | - | 衡量信度 |
| **中介效应** | 间接影响 | - | 路径分析 |

## 🔍 如何选择统计方法

### 比较均值

```
需要比较均值？
├─ 一个组 → 单样本t检验
├─ 两个组
│  ├─ 独立 → 独立样本t检验
│  └─ 配对 → 配对样本t检验
└─ 多个组 → 单因素ANOVA
```

### 分析关系

```
需要分析关系？
├─ 两个变量的线性关系 → Pearson相关
├─ 多个变量的关系 → 相关矩阵
├─ 预测因变量 → 回归分析
│  ├─ 一个自变量 → 一元回归
│  └─ 多个自变量 → 多元回归
└─ 间接影响 → 中介效应
```

### 选择图表

```
需要展示什么？
├─ 分类数据比较 → 柱状图
├─ 两变量关系 → 散点图
├─ 时间序列 → 折线图
├─ 数据分布 → 箱线图或直方图
├─ 比例关系 → 饼图
└─ 多变量关系 → 热力图
```

## 📋 常用术语解释

### P值解读

- **P < 0.001** (极显著) → **\*\*\*** → 非常强的证据
- **P < 0.01** (非常显著) → **\*\*** → 很强的证据
- **P < 0.05** (显著) → **\*** → 有证据
- **P ≥ 0.05** (不显著) → **ns** → 无证据

### 效应量解读

| 指标 | 小效应 | 中等效应 | 大效应 |
|------|--------|---------|--------|
| Cohen's d | 0.2 | 0.5 | 0.8 |
| 相关系数r | 0.1 | 0.3 | 0.5 |
| R² | 0.01 | 0.06 | 0.14 |

### 相关系数强度

- **\|r\| > 0.7** → 强相关
- **0.5 < \|r\| ≤ 0.7** → 中等相关
- **0.3 < \|r\| ≤ 0.5** → 弱相关
- **\|r\| ≤ 0.3** → 很弱相关

## 💡 实用建议

### 报告结果时

✅ **要做**：
- 报告具体的统计量（如t=2.5）
- 报告P值（如p=0.02）
- 报告效应量（如d=0.6）
- 报告置信区间（如95% CI: [70,80]）

❌ **不要做**：
- 只说"显著"或"不显著"
- 忽视效应量
- 忽视样本量
- 过度解释结果

### 选择方法时

✅ **要做**：
- 检查数据是否满足前提条件
- 考虑样本量
- 根据研究问题选择方法
- 报告所有相关统计量

❌ **不要做**：
- 盲目选择方法
- 忽视前提条件
- 多重比较不校正
- 只报告显著结果
