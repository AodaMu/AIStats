## 📖 页面内容

### 5个标签页

#### 1️⃣ 📊 描述统计
解释数据基本特征的术语：
- **均值** - 所有数据的平均值
- **标准差** - 衡量数据离散程度
- **中位数** - 排序后的中间值（50%分位数）
- **Q1** - 25%分位数
- **Q3** - 75%分位数
- **最大值** - 数据中的最大数值
- **最小值** - 数据中的最小数值

**可视化示例**：
- 直方图 - 展示数据分布，标记均值
- 箱线图 - 展示四分位数和中位数

#### 2️⃣ 🔬 假设检验
解释统计显著性检验的术语：
- **P值** - 概率值，P<0.05表示显著
- **t值** - t检验的统计量
- **显著性** - 显著性水平标记（*/**/**/**）
- **自由度** - 可以自由变化的数据个数
- **置信区间** - 真实参数所在的范围
- **效应量** - 差异的实际大小
- **Cohen's d** - 效应量的标准化指标

**可视化示例**：
- t分布图 - 显示显著性区域
- 两组差异对比 - 展示效应量

#### 3️⃣ 🔗 相关与回归
解释变量关系的术语：
- **相关系数(r)** - 衡量两变量关系强度
- **相关矩阵** - 多个变量的相关系数表格
- **显著性检验** - 检验相关是否显著
- **回归方程** - 用于预测的数学方程
- **R²** - 模型解释变异的比例
- **调整R²** - 考虑自变量个数的R²
- **VIF** - 检测多重共线性的指标

**可视化示例**：
- 散点图 - 展示变量关系和趋势线
- 热力图 - 展示相关矩阵

#### 4️⃣ 📈 图表类型
解释各种图表的用途：
- **柱状图** - 比较不同类别的数值
- **散点图** - 展示两个变量的关系
- **折线图** - 显示数据随时间的变化
- **箱线图** - 展示数据分布和异常值
- **饼图** - 展示各部分占整体的比例
- **直方图** - 展示连续数据的分布
- **热力图** - 用颜色表示数值大小

**可视化示例**：
- 柱状图示例 - 班级平均成绩对比
- 饼图示例 - 班级人数占比
- 散点图示例 - 学习时间与成绩关系
- 折线图示例 - 时间序列趋势

#### 5️⃣ 🎯 高级分析
解释高级统计方法：
- **单因素ANOVA** - 比较多个组的均值
- **Levene检验** - 检验方差齐性
- **Cronbach's Alpha** - 衡量量表内部一致性
- **中介效应** - 研究间接影响
- **独立样本t检验** - 比较两个独立组
- **配对样本t检验** - 比较同一组的前后差异
- **单样本t检验** - 检验样本均值

**对比表**：
- 检验方法对比表 - 显示各种检验的适用场景

## 🔍 搜索功能

### 如何使用搜索

1. **打开术语解释页面**
   - 点击左侧导航栏的"📚 术语解释"

2. **输入搜索关键词**
   - 在搜索框中输入术语名称
   - 支持模糊搜索（如输入"均"会找到"均值"）

3. **实时过滤**
   - 搜索结果会自动过滤显示
   - 只显示匹配的术语

4. **查看详细信息**
   - 点击术语的展开按钮查看详细解释
   - 包含定义、公式、例子等

### 搜索示例

| 搜索词 | 会找到 |
|-------|-------|
| 均值 | 均值 |
| 标准 | 标准差 |
| p | P值 |
| 相关 | 相关系数、相关矩阵、相关性检验 |
| 图表 | 柱状图、散点图、折线图等 |
| 检验 | 所有检验方法 |
