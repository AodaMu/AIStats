## 📋 对比表（4个）

### 1. 检验方法对比表
| 检验方法 | 比较对象 | 样本数 | 前提条件 | 适用场景 |
|---------|---------|--------|---------|---------|
| 单样本t检验 | 样本 vs 总体 | 1个 | 数据正态 | 检验平均值 |
| 配对样本t检验 | 同组前后 | 1个（2次测量） | 差值正态 | 重复测量 |
| 独立样本t检验 | 两个独立组 | 2个 | 正态+方差齐 | 组间比较 |
| 单因素ANOVA | 多个独立组 | 3个或以上 | 正态+方差齐 | 多组比较 |

### 2. Cronbach's Alpha 信度评价表
| Alpha值 | 信度评价 | 建议 |
|---------|---------|------|
| α ≥ 0.9 | 优秀 | ✅ 可用 |
| 0.8 ≤ α < 0.9 | 很好 | ✅ 可用 |
| 0.7 ≤ α < 0.8 | 可接受 | ✅ 可用 |
| 0.6 ≤ α < 0.7 | 一般 | ⚠️ 谨慎使用 |
| α < 0.6 | 不可接受 | ❌ 不可用 |

### 3. Cohen's d 效应量表
| 效应量 | 效应大小 | 实际意义 |
|--------|---------|---------|
| d = 0.2 | 小效应 | 差异较小 |
| d = 0.5 | 中等效应 | 差异中等 |
| d = 0.8 | 大效应 | 差异明显 |
| d > 1.0 | 非常大效应 | 差异非常明显 |

### 4. R² 解释力表
| R²值 | 解释力 | 评价 |
|------|--------|------|
| R² = 0.01 | 1% | 很小 |
| R² = 0.06 | 6% | 小 |
| R² = 0.14 | 14% | 中等 |
| R² > 0.20 | >20% | 大 |

### 如何使用
