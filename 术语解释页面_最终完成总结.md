## 🚀 使用方式

### 启动应用
```bash
streamlit run app.py
```

### 访问术语解释页面
1. 打开应用
2. 点击左侧导航栏"📚 术语解释"
3. 选择要查看的标签页
4. 查看详细的术语解释和可视化

### 搜索功能
1. 在搜索框输入术语名称
2. 实时显示搜索结果
3. 点击"查看"自动跳转到对应标签页
